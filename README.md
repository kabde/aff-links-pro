# Aff Links Pro

Plugin WordPress pour gerer des liens d'affiliation avec:

- CPT `aff`.
- URLs publiques `/go/slug`.
- Redirection vers l'URL d'affiliation.
- Colonne QR code dans l'administration.
- Shortcode `[aff_link id="123"]` ou `[aff_link slug="amazon"]`.

## Installation

1. Copier le dossier `aff-links` dans `/wp-content/plugins/`.
2. Activer `Aff Links Pro` dans WordPress.
3. Aller dans `Affiliations` pour creer les liens.

## Version

Version: `1.2.1`
